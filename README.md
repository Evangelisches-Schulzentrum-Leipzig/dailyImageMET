# Daily Signage Image Generator 
This webpage selects an daily image from a list of MET objects ids fetches the image data and URL and renders everything for a fullscreen signage application. 

## Features
- Fullscreen image display with blurred background
- QR-Code to MET detail page
- Description with author, title and year 

## Requirements
- Browser (in signage software e.g. [Xibo](https://xibosignage.com/))
